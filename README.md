# X-Input
Se trata de um construtor de inputs usando javascript puro. Nele você poderá passar os seguintes atributos:

| Atributo      | Obrigatório | Valor padrão |
|---------------|-------------|--------------|
| contexto      | ❌           | text padrão  |
| x-name        | ❌           |              |
| x-id          | ✅           |              |
| x-class       | ❌           |              |
| x-placeholder | ❌           |              |
